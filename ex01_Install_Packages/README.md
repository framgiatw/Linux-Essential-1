## Working with `dpkg`
Find the usage of the following commands.
- `dpkg -l`
- `dpkg -l {package_name}`
- `dpkg -L {package_name}`
- `dpkg -i {package_name}`
- `dpkg -r {package_name}`
- `dpkg -P {package_name}`
- `dpkg -c {package_name}`
- `dpkg -S {/path/to/file}`
